#+title: Pablo's dotfiles
#+author: Pablo Stafforini

This repository contains my dotfiles, including

- my [[file:emacs/config.org][Emacs config]];
- my [[file:agents/README.org][agent configuration synchronization docs]];
- my [[file:karabiner/modifications.org][Karabiner modifications]]; and
- my [[file:macos/defaults.org][macOS defaults]].

* Skills

- =config-audit=: Audits agent configuration for redundancy, conflicts, dead weight, staleness, and instruction drift.
- =codex-app-browser=: Maintains the Codex App browser plugin setup for local browser-use workflows.
- =find-skills=: Discovers, vets, and optionally installs reusable agent skills.
- =gitguardian-triage=: Triages and remediates GitGuardian secret incidents.
- =install-mcp-server=: Installs and configures MCP servers with auth verification, placement, registration, sync, and restart/testing checks.
- =move-session-log=: Relocates Codex session logs by rewriting embedded project paths.
  Copied =SKILL (1).md= variants, when present, are kept paired across the
  project-local Claude and Codex skill trees.
- =open-session-log=: Programmatic-only helper that opens the current Claude Code or Codex session log in Emacs via =agent-log=.
- =optimize-agent-instructions=: Optimizes project agent instruction files.
- =release-dotfiles=: Prepares and publishes dotfiles releases, including
  checks/restoration for temporary Elpaca PR fork pins in =emacs/config.org=.
- =skill-audit=: Audits or improves agent skills for trigger quality, workflow design, verification, portability, and maintainability.
- =skill-creator=: Creates or iteratively improves agent skills.
- =skill-discovery=: Scans recent sessions to discover repeated workflows worth turning into skills.
- =skill-resolver=: Resolves and reads agent skill files by skill name.
