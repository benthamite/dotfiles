This manual describes the features and customization options for the Emacs Lisp file =kmacro-extras.el=.

* Overview
:PROPERTIES:
:CUSTOM_ID: h:overview
:END:

=kmacro-extras.el= extends Emacs' built-in keyboard macro system
(=kmacro.el=).  The stock macro counter can insert sequential numbers
(1, 2, 3, ...) during macro execution, but switching to alphabetical
sequences (a, b, c, ...) requires manually changing the counter format
and resetting the counter value---two separate operations that are easy
to forget.  This package wraps both steps into a single toggle command.

* Commands
:PROPERTIES:
:CUSTOM_ID: h:commands
:END:

** Toggling the counter format
:PROPERTIES:
:CUSTOM_ID: h:counter-toggle
:END:

#+findex: kmacro-extras-counter-toggle-alpha-number
The command ~kmacro-extras-counter-toggle-alpha-number~ switches the
keyboard macro counter between numeric and alphabetical mode:

- When the current format is numeric (=%d=), the command sets the
  format to =%c= (character) and resets the counter to 97, the ASCII
  code for =a=.  Subsequent insertions via ~kmacro-insert-counter~
  will produce =a=, =b=, =c=, and so on.
- When the current format is alphabetical (=%c=), the command sets it
  back to =%d= and resets the counter to 1.

This is especially handy when you need to label items with letters in
one macro and switch back to numbers for the next, without
interrupting your editing flow.  A confirmation message (="Set to
alphabetical"= or ="Set to numeric"=) is displayed in the echo area
after each toggle.
