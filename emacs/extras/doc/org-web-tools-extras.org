This manual describes the features and customization options for the Emacs Lisp file =org-web-tools-extras.el=.

* Overview
:PROPERTIES:
:CUSTOM_ID: h:overview
:END:

=org-web-tools-extras.el= extends =org-web-tools=, the package that provides utilities for interacting with web content within Org mode. The upstream package offers commands for inserting Org-formatted links and reading web pages, but some of these are designed for interactive use and do not integrate smoothly with programmatic contexts such as Org Capture templates.

=org-web-tools-extras.el= addresses this gap with two focused functions:

- *Link insertion from the kill ring.* A function that fetches the title for a URL found in the kill ring and returns a formatted Org link, suitable for use in Org Capture templates ([[#h:insert-link-clipboard][Inserting a link from the clipboard]]).
- *Title extraction.* A function that fetches and returns just the title of a web page at a given URL, using the =plz= HTTP library for retrieval ([[#h:title-for-url][Extracting a page title]]).

The package depends on =org-web-tools= and =paths=.

* Functions
:PROPERTIES:
:CUSTOM_ID: h:functions
:END:

** Inserting a link from the clipboard
:PROPERTIES:
:CUSTOM_ID: h:insert-link-clipboard
:END:

#+findex: org-web-tools-extras-insert-link-for-clipboard-url
The function ~org-web-tools-extras-insert-link-for-clipboard-url~ retrieves the first URL from the kill ring (using ~org-web-tools--get-first-url~), fetches the page title for that URL, and returns a fully formatted Org link in the form =[[url][title]]=. It delegates the actual link construction to the internal ~org-web-tools--org-link-for-url~ function.

This function is designed to be called from an Org Capture template. When you copy a URL in a browser and invoke an Org Capture template that uses this function, it automatically creates a properly formatted link without requiring you to type the title manually. For example, you might use it in a capture template like this:

#+begin_src emacs-lisp
(setq org-capture-templates
      '(("l" "Link" entry (file "~/links.org")
         "* %(org-web-tools-extras-insert-link-for-clipboard-url)\n")))
#+end_src

** Extracting a page title
:PROPERTIES:
:CUSTOM_ID: h:title-for-url
:END:

#+findex: org-web-tools-extras-org-title-for-url
The function ~org-web-tools-extras-org-title-for-url~ fetches the HTML content of a given URL, extracts the =<title>= element, and returns the cleaned title string. It uses the =plz= library to perform the HTTP request and ~libxml-parse-html-region~ to parse the HTML DOM.

If the URL argument is omitted, the function defaults to the first URL found in the kill ring, making it convenient for quick programmatic lookups. If the HTTP request fails, the HTML cannot be parsed, or the page has no =<title>= element, the function returns the fallback string ="Downloaded webpage"= rather than signaling an error. Title cleanup is performed by ~org-web-tools--cleanup-title~, which strips extraneous whitespace and normalizes the string for use in Org links.
